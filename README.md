# Aiml Java Interpreter
AIML 2.0 Interpreter for Java

## Build
cd ./scripts && ./checkout.sh && cd -
### Gradle
gradle clean build

### Maven
mvn clean package

## Bots
https://github.com/AIMLang/aiml-bots.git

## Dependencies
- Spring Boot (1.5.3.RELEASE)
- Logback (1.2.3)
- Slf4j (1.7.25)
- JUnit (4.12)
