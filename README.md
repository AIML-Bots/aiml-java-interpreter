# Aiml Java Interpreter
AIML 2.0 Interpreter for Java

## Build
cd ./scripts && ./checkout.sh && cd -

gradle clean build

## Bots
https://github.com/AIML-Bots/aiml-bots.git

## Dependencies
- Logback (1.2.3)
- Slf4j (1.7.25)
- Lombok (1.16.16)
- Gradle (3.5)
- JUnit (4.12)
- JaCoCo (0.7.9)
